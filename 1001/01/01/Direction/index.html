自定义数学符号

不要直接修改 _config.yml, 在 macros.json 里添加, 会自动注入 (LoadMacros.js).

自定义字体

先在 source\fonts 里保存字体, 然后在 %THEME%\source\css\font.styl (or style.styl) 里加上 @font-face 的声明.

然后使用 [自定义字体文本]{style="font-family: 'MyFont'"} 即可.

另外, 上述语法都支持设置字体大小, 字重等, []{} 为 markdown-it-attrs 提供, 可以直接注入 CSS 属性.

PS: 在展示 {% ... %} 时, 请记得最好放在 {% raw %}{% endraw %} 里, 防止发生不匹配的错误, 另外, 请注意 {% ... %} 无法嵌套使用以及一些神秘细节, 比如在行内代码 `` 中 {% raw %}{% endraw %} 应该放在里面, 而行间代码块 ``` 应该放在外面.

PPS: 其实 {% raw %}{% endraw %} 才是最难实现的那个, 试了许多方法, 最后可行的是 <code>{<span></span>% raw %<span></span>}</code>, 感觉有点抽象.

Post 元信息

  • breaks: true/false: 默认为 true, 控制文章的书写模式.
  • featured_image: url/[url, text, link]: url: 头图链接; text: 显示在图片正下方的文本, 一般放作者信息; link: 存在时 text 变为超链接; 默认为一条分割线.
  • title_style: "css-style": 设置标题显示的字体.
  • list_title_style: "css-style": 设置标题在列表里显示的字体.
